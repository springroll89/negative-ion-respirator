用户
扫码
→
APP
身份验证
发起启动
→
Go API
鉴权/计费
下发指令
→
EMQX
MQTT转发
到设备Topic
→
ESP32
启动设备
温控/状态上报
返回链路: ESP32 → MQTT → Go API → APP (使用结束/状态上报/费用结算)

Topic 方向 说明 QoS

device/{id}/ota 云→设备 固件OTA升级 2
device/{id}/heartbeat 设备→云 心跳/在线状态 0
device/{id}/event 设备→云 异常告警/故障事件 1
