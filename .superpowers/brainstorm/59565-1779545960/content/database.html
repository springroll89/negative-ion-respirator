核心表说明

表名 存储引擎 说明 预估数据量
usersPG用户账户、余额万级
devicesPG设备注册信息、在线状态千级
device_configPG单设备温度参数千级
region_configPG按地区/季节的默认配置模板百级
ordersPG使用订单、计费记录十万级
device_logsTimescaleDB设备遥测数据(温度/离子/事件)千万级
batch_tasksPG批量操作任务(调参/升级)百级
admin_usersPG后台管理员十级

TimescaleDB 超表设计 (device_logs)

-- 创建超表，按时间自动分区
SELECT create_hypertable('device_logs', 'timestamp', chunk_time_interval => INTERVAL '1 day');

-- 自动压缩 7 天前的数据
SELECT add_compression_policy('device_logs', INTERVAL '7 days');

-- 自动删除 365 天前的数据
SELECT add_retention_policy('device_logs', INTERVAL '365 days');
