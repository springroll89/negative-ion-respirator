# 负氧离子呼吸器 — 硬件电路设计

## 1. 系统框图

```
220V AC ──→ 电源适配器 ──→ 12V/5V DC
                              │
                    ┌─────────┼─────────┐
                    │         │         │
                    ▼         ▼         ▼
               ESP32       加热管     负离子
              主控MCU      (PWM)     发生器
                │           │          │
       ┌────────┼───────┐   │          │
       │        │       │   │          │
       ▼        ▼       ▼   ▼          ▼
    DS18B20  LED×2   SIMCom   MOSFET   继电器
    温度传感  指示灯   4G模块   驱动     控制
```

## 4. 关键电路说明

### 4.1 加热管 PWM 驱动

```
ESP32 GPIO25 ──→ R1 (1kΩ) ──→ MOSFET IRF520 Gate
                                │
                    12V ──→ 陶瓷加热管 ──→ MOSFET Drain
                                │
                              GND ←── MOSFET Source

保护: MOSFET Gate 对地并 10kΩ 下拉电阻
      加热管两端并续流二极管 (1N4007)
```

- 加热管规格: 12V / 50W, PTC 陶瓷
- MOSFET: IRF520 (N-Channel, 100V/9.2A)
- PWM 频率: 1kHz, 占空比 0-100%
- 软件硬上限: 80°C (占空比限幅)
- 硬件保护: >85°C 触发 GPIO14 断开继电器

### 4.2 温度传感 (NTC + 分压)

```
3.3V ──→ R2 (10kΩ 精密 1%) ──┬──→ ESP32 ADC (GPIO36/GPIO39)
                              │
                            NTC (10kΩ @25°C, B=3950)
                              │
                             GND

Vout = 3.3V × R_NTC / (R_NTC + 10kΩ)
```

ADC 量程: 0-3.3V, 12bit (0-4095)
需要校正表将 ADC 值转换为实际温度。
参考: NTC 10kΩ B3950, 测温范围 -20~105°C

### 4.3 负离子发生器控制

```
ESP32 GPIO26 ──→ 继电器模块 IN1 ──→ 12V 继电器
                                      │
                        12V ──→ COM ──→ NO ──→ 负离子发生器 VCC
                                      │
                                    NC (空)

ESP32 GPIO27 ←── FAULT 检测 ←── 负离子发生器
```

负离子发生器规格:
- 输入: 12V DC, <10W
- 输出浓度: ≥30 万个/cm³
- 带自检 FAULT 信号输出 (开漏, 上拉 10kΩ)

### 4.4 硬件过温保护

这是一个**独立于 MCU 的硬件保护回路**：

```
温度开关 (85°C 常闭) ──→ 继电器 IN2 ──→ 常闭触点
    (贴于加热管表面)          │
                          COM → 加热管电源
                          NC → 12V 供电
                          
正常工作: 温度 <85°C, 开关闭合, 继电器 NC 导通, 加热管可工作
过温保护: 温度 ≥85°C, 开关断开, 继电器 NC 断开, 加热管断电

MCU 监控: ESP32 GPIO14 控制继电器 IN2, 软件也可主动断电
```

### 4.5 供电设计

```
220V AC ──→ 12V/5A 适配器 ──→ 12V DC 母线
                                │
              ┌─────────────────┼───────────────┐
              │                 │               │
              ▼                 ▼               ▼
          陶瓷加热管        负离子发生器    5V 降压模块
          (12V/4A)         (12V/500mA)    (LM2596)
                                              │
                              ┌───────────────┤
                              │               │
                              ▼               ▼
                         3.3V LDO        SIMCom 4G
                       (AMS1117-3.3)    (3.8V/2A 独立)
                              │
                              ▼
                           ESP32
                          (3.3V/500mA)
```

- 总功率: 约 60W (加热管 50W + 负离子 10W + 控制电路)
- 适配器: 12V/5A (60W)，需通过 CCC 认证

## 6. PCB 布局建议

- 加热管驱动回路（12V/4A）走线宽度 ≥2mm
- MOSFET 加装散热片
- NTC 传感线与功率线分开布局，避免干扰
- NTC1 (加热管温度) 紧贴加热管表面固定
- NTC2 (出气口温度) 置于出气口风道内
- ESP32 与 4G 模块保持 2cm 以上距离
- 温度开关固定于加热管侧面，用导热硅脂
- 12V/5V/3.3V 电源分区，数字/模拟地分离，单点接地
